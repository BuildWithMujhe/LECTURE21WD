# CSS Media Queries and Custom Variables

This repository contains my practice work and notes from **Lecture 21** of the Web Development Course. In this lecture, I explored CSS Media Queries and Custom Variables, two powerful concepts used to create responsive, scalable, and maintainable web designs.
